# @sittir/rust

134 Rust IR node kinds with fluent builders, render pipeline, and validation — all generated from the tree-sitter-rust grammar by `@sittir/codegen`.

## Installation

```bash
pnpm add @sittir/rust
```

## Quick Start

```ts
import { ir } from '@sittir/rust';

const source = ir.file().children([
  ir.use('std::collections::HashMap').build(),
  ir.structItem('Config').body('name: String,').build(),
  ir.fn('main').body('println!("hello");').build(),
]).render();
```

Output:

```rust
use std::collections::HashMap;

struct Config {
    name: String,
}

fn main() {
    println!("hello");
}
```

## Generated API

Everything below is generated by `@sittir/codegen` from `tree-sitter-rust/src/node-types.json`. The 134 node builders, fluent namespace, types, and render cases are all codegen output — not hand-written.

### Fluent Builders (`ir.*`)

The `ir` namespace exposes all 134 node kinds as fluent builder functions:

```ts
ir.fn('main')              // function_item
ir.structItem('Config')    // struct_item
ir.use('std::io')          // use_declaration
ir.impl('Config')          // impl_item
ir.if('x > 0')             // if_expression
ir.macro_invocation(...)   // macro_invocation
ir.file()                  // source_file
ir.enum('Color')           // enum_item
ir.trait('Display')        // trait_item
ir.match('value')          // match_expression
// ... all 134 node kinds
```

Each builder returns a fluent chain with `.build()`, `.render()`, and `.renderSilent()`.

### Render + Validate

```ts
import { render, renderSilent, validate, assertValid } from '@sittir/rust';

// Render with validation (throws on error)
const source = render(node);

// Render without validation
const source = renderSilent(node);

// Standalone validation
const result = validate('fn main() {}');  // { ok: true }
assertValid('fn main() {');               // throws with error details
```

## How It's Generated

```
tree-sitter-rust/src/node-types.json
        |
  @sittir/codegen
        |
  ├── src/nodes/*.ts      134 node builder files (factory + fluent class)
  ├── src/fluent.ts        ir namespace re-exporting all builders
  ├── src/types.ts         Grammar-derived TypeScript types
  └── src/render.ts        Render switch over all 134 node kinds
```

Only `index.ts`, `render-valid.ts`, `validate-fast.ts`, and `grammar.ts` are hand-maintained.
